## 🚀 使用方法

### 1. 在微信开发者工具中打开

1. 打开微信开发者工具
2. 选择「导入项目」
3. 项目目录选择：`/workspace/dist/`
4. AppID：填入您的真实小程序 AppID（或使用测试号）
5. 项目名称：海豚电竞

### 2. 修改 API 域名配置

在微信开发者工具中：
- 进入「详情」→「本地设置」
- 勾选「不校验合法域名、web-view（业务域名）、TLS 版本以及 HTTPS 证书」（开发阶段）

或者在代码中修改 API 地址：
- 在 `/workspace/src/network.ts` 中修改 `baseURL` 为您的服务器地址

### 3. 服务器配置

确保后端服务已启动：
```bash
cd /workspace/server && pnpm dev
```

默认端口：`http://localhost:3000`

## 🔧 常见问题

### Q: 打包后白屏？
A: 检查 `project.config.json` 中的 `miniprogramRoot` 是否正确指向 `./`

### Q: API 请求失败？
A: 1. 检查后端服务是否启动
   2. 微信开发者工具是否勾选「不校验合法域名」
   3. network.ts 中的 baseURL 是否正确

### Q: 图片上传功能无法使用？
A: 确保后端上传模块已正确配置，检查 API 路径是否匹配

### Q: 如何上传到微信公众平台？
A: 1. 在微信开发者工具中点击「上传」
   2. 填写版本号和项目备注
   3. 登录微信公众平台提交审核

## 📝 注意事项

1. **域名配置**：正式发布前，需要在微信公众平台配置合法域名
2. **AppID**：测试时可使用测试号，正式发布需要真实 AppID
3. **服务器地址**：确保 `PROJECT_DOMAIN` 环境变量配置正确
4. **权限检查**：不同角色的权限已在代码中严格控制
